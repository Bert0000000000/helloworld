# 用户登录功能说明

## 功能概述

HelloWorld 应用已集成完整的用户登录系统，支持用户注册、登录、认证等功能。

## 技术架构

### 后端技术栈
- **框架:** Flask 3.0.0
- **ORM:** SQLAlchemy
- **数据库:** SQLite
- **认证:** JWT (JSON Web Token)
- **加密:** bcrypt (12 轮盐值)

### 前端技术栈
- **框架:** React 18
- **路由:** React Router v6
- **UI:** Ant Design 5.x
- **状态:** localStorage (Token 存储)

## 数据库设计

### users 表结构

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| username | String(80) | 用户名（唯一） |
| email | String(120) | 邮箱（唯一） |
| password_hash | String(255) | 密码哈希 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |
| is_active | Boolean | 账号状态 |

## API 接口

### 1. 用户注册

**POST** `/api/auth/register`

**请求体:**
```json
{
  "username": "admin",
  "email": "admin@example.com",
  "password": "123456"
}
```

**响应:**
```json
{
  "code": 200,
  "data": {
    "user": {
      "id": 1,
      "username": "admin",
      "email": "admin@example.com",
      "created_at": "2026-04-17T02:30:00",
      "is_active": true
    },
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  },
  "message": "注册成功"
}
```

### 2. 用户登录

**POST** `/api/auth/login`

**请求体:**
```json
{
  "username": "admin",
  "password": "123456"
}
```

**响应:**
```json
{
  "code": 200,
  "data": {
    "user": { ... },
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  },
  "message": "登录成功"
}
```

### 3. 获取用户信息

**GET** `/api/auth/profile`

**请求头:**
```
Authorization: Bearer <token>
```

**响应:**
```json
{
  "code": 200,
  "data": {
    "id": 1,
    "username": "admin",
    "email": "admin@example.com",
    "created_at": "2026-04-17T02:30:00",
    "is_active": true
  },
  "message": "success"
}
```

### 4. 验证 Token

**POST** `/api/auth/verify`

**请求体:**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

## 安全特性

### 密码加密
- 使用 bcrypt 算法
- 12 轮盐值生成
- 密码不以明文存储

### JWT 认证
- Token 有效期：7 天
- HS256 签名算法
- 包含 user_id 和 username

### 输入验证
- 用户名：必填，不能为空
- 邮箱：必填，格式验证
- 密码：至少 6 位

### 错误处理
- 用户不存在：401
- 密码错误：401
- 账号禁用：401
- Token 无效：401
- Token 过期：401

## 前端功能

### 登录页面
- 位置：`/login`
- 组件：`src/pages/Login.jsx`
- 功能：
  - 用户名/邮箱登录
  - 表单验证
  - 错误提示
  - 加载状态

### 路由保护
- 未登录用户自动跳转到登录页
- Token 存储在 localStorage
- 用户信息持久化

### 用户菜单
- 显示当前用户名
- 退出登录功能
- 清除本地存储

## 快速开始

### 1. 启动后端

```bash
cd backend
pip install -r requirements.txt
python app.py
```

后端服务：http://localhost:5000

### 2. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端服务：http://localhost:5173

### 3. 测试账号

首次启动时数据库为空，可通过以下方式创建测试账号：

**方式一：使用注册功能（开发中）**

**方式二：直接插入数据库**
```python
# 在 Python shell 中执行
from backend import app, db, User
from auth import generate_password_hash

with app.app_context():
    user = User(
        username='admin',
        email='admin@example.com',
        password_hash=generate_password_hash('123456')
    )
    db.session.add(user)
    db.session.commit()
```
